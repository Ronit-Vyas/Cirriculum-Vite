# 📄 My Cirriculum Vite 

A clean and responsive Resume Website built using HTML, CSS and JavaScript.

## ✨ Features

- Professional Resume UI
- Responsive Design
- Skills Section
- Education Table
- Projects Section
- Contact Links
- Download Resume as PDF
- Modern Animations

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- html2pdf.js
